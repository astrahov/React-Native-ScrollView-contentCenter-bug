# Getting Started

```sh
`npm ci`
```

## iOS

```sh
npm run ios:setup:bundle-install
```

### Pods

```sh
npm run ios:setup:pod-install
```

or

```sh
npm run ios:setup:pod-install--with-repo-update
```

# Step 1: Start Metro

```sh
npm run server:start--with-reset-cache
```

# Step 2: Build and run your app

### Android

```sh
npm run android
```

### iOS

```sh
npm run ios
```
